## Scope of Fork

CodexBar Lite keeps CodexBar UI/features relevant to Codex/Claude while replacing data acquisition with a strict lightweight policy:

- local credentials + direct API calls only
- no WebView scraping
- no browser cookie import/decryption
- no PTY/RPC status probing
